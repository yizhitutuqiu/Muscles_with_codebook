## 消融实验（Ablation）细节总结

本次批量消融实验（`batch_stage2_h5`）共包含 4 组子实验，各自相对于 Baseline 的设计如下：

1. **`exp_h2_baseline` (基线模型)**:
   - **设计**: 采用最初的原始网络架构（原版 DSTFormer + 原版 DCSA 融合模块），不包含任何 H5 提出的结构改动。
   - **作用**: 作为严格的控制变量基线，反映模型在缺乏时空归纳偏置时的 `no_cond` 性能下限（10.96）。

2. **`exp_h5_no_ltc` (无局部时序卷积)**:
   - **设计**: 在基线上增加了**离散 Token 时空位置编码**、**旋转位置编码 (RoPE)** 以及 **SwiGLU 门控机制**，但**特意去除了**时序前馈网络中的局部时序卷积 (Local Temporal Convolution, LTC)。
   - **作用**: 验证 LTC 在强制模型学习 EMG 连续信号局部平滑性方面的不可替代性。结果表明，即使没有 LTC，仅靠 RoPE 和离散时空编码也能将误差大幅降至 10.50。

3. **`exp_h5_no_rope` (无旋转位置编码)**:
   - **设计**: 在基线上增加了**离散 Token 时空位置编码**、**LTC** 以及 **SwiGLU 门控机制**，但**特意去除了**自注意力机制中的旋转位置编码 (RoPE)。
   - **作用**: 验证 RoPE 在长序列周期性相对运动（如深蹲、跑步）建模上的价值。结果同样降至 10.50，说明 LTC 和离散编码也贡献了巨大的泛化增益。

4. **`exp_h5_full` (H5 完全体)**:
   - **设计**: 集成了所有 H5 改进方案（离散 Token 时空 PE + RoPE + LTC + SwiGLU）。
   - **作用**: 证明各项底层时空归纳偏置相辅相成，组合后能最大化提升模型在 `no_cond` 场景下的空间对齐与时序泛化能力，达到最优性能（10.49）。

## H6 (结构分支优化) 与 Stage 1 软切分 (CIF)

在突破了 H5 的 10.5 瓶颈后，我们尝试从结构分支和数据表示层面进一步提升模型能力，目标是逼近 RMSE 9.x。

我们利用 `batch_train.stage2_h6.yaml` 进行了一组平行的 H6 消融实验，结果如下：

| 模型 / 架构配置 | 核心机制 | 平均 RMSE | 结论 |
| :--- | :--- | :---: | :--- |
| `exp_h6_b_stgcn` | 时空图卷积网络 (ST-GCN) | 11.34 | **性能退化**。直接将图结构引入连续信号可能破坏了时序平滑性。 |

### Stage 1 动态语义软切分 (CIF)

- **动机**：传统的 TCN 或自适应池化采用固定长度或固定频率进行切分，无法自适应人体运动速度和复杂度的变化。
- **机制**：我们引入了语音识别中经典的 **Continuous Integrate-and-Fire (CIF)** 机制。该机制通过预测每帧的权重（$\alpha$），并在时间轴上累加，当累加值达到阈值时便“触发（Fire）”生成一个具有宏观时间感受野的语义 Token。
- **验证成果**：
  - Stage 1 的重构误差 (RMSE) 降至 **17.20**。
  - Codebook 激活率极高：骨骼分支 (`j3d_active_rate`) 达到 **83.9%**，肌电分支 (`emg_active_rate`) 达到 **41.0%**，远超之前固定频率切分的利用率。
  - 这表明模型成功学到了具有高度代表性的运动学词汇表。

### 未来展望 (H7 实验)
鉴于 **H6-C (MoE)** 和 **Stage 1 (CIF)** 各自在其领域取得了最优表现，目前已正式启动 **H7 实验** (`exp_h7_cif_moe`)，旨在将“动态语义软切分 (CIF)”与“动态计算路由 (MoE)”结合，期待这两种自适应机制能够相互成就，实现更极致的泛化性能。

## H7：双向对称跨注意力 (Symmetric DCSA) 与 双分支并行时空融合 (Dual-Branch MoE)

为了进一步挖掘时空特征的交互潜力，我们在 H6 MoE 的基础上，回顾并引入了经典 MotionBERT 的**并行双分支架构**（先空间后时间 + 先时间后空间，最后加权融合），同时将原先以连续特征为 Query、离散特征为 Key/Value 的单向跨注意力机制（Asymmetric DCSA），升级为**双向对称跨注意力机制（Symmetric DCSA）**。

### H7 消融实验结果

我们在 GPU 4, 5, 6 上通过 `run_h7_symmetric_dual.yaml` 进行了三组消融实验，评估这两种机制在无条件 (`no_cond`) 下对泛化性能的贡献：

| 模型 / 架构配置 | 核心机制 | 平均 RMSE | 结论 |
| :--- | :--- | :---: | :--- |
| `exp_h7_sym_dcsa_only` | 仅使用 Symmetric DCSA (串行 MoE 主干) | 10.55 | **持平基线**。与官方 `cond=True` 性能持平，但相比 H6 MoE 的 10.39 有所退化，说明单纯的对称融合在串行架构下未发挥优势。 |
| `exp_h7_dual_branch_only` | 仅使用 双分支并行 MoE 主干 (Asymmetric DCSA) | 10.42 | **显著提升**。证明了空间-时间与时间-空间双路并行的特征提取方式，比单一的串行提取更适合复杂的肌电预测任务。 |
| **`exp_h7_full_sym_dual`** | **Symmetric DCSA + 双分支并行 MoE** | **10.30** | **突破性进展**。双向对称跨注意力与并行双分支主干产生了极佳的化学反应，将误差大幅降低至 10.30，为目前无条件泛化任务的最佳成绩。 |

### 结论与 Paper Storyline 补充

H7 的实验结果为我们的论文故事线补上了关键的一块拼图：
1. **连续与离散特征的平等对话**：通过 Symmetric DCSA，我们证明了不仅连续特征需要查询离散运动学字典（骨骼查字典），离散特征也需要锚定当前的连续物理状态（字典查骨骼）。
2. **多维时空视角的互补**：双分支并行架构（ST 路线与 TS 路线）结合 Token-level MoE 路由，让模型能够同时捕捉“特定关节在全时间段的运动规律”和“特定时刻全身关节的协同发力”，这两种视角的融合是逼近真实肌电信号的关键。
