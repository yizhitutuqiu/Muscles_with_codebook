# 实验报告 (2026-04-23)

## 核心进展：H5 架构重构与泛化性能突破

通过从网络架构底层引入真正的时空归纳偏置，我们在完全不输入身份条件 (`no_cond`) 的情况下，成功将 Stage 2 (Pose2EMG) 的平均 RMSE 误差降至 **10.49**，不仅达成了预定目标（9~10的区间边界），并且首次超越了官方带有身份条件 (`cond=True`) 的基线模型上限（10.55）。

## 实验结果与指标

以下为基于验证集（ID_nocond_exercises）的各项架构改动消融实验结果（Average RMSE）：

| 模型 / 架构配置 | 测试模式 | 平均 RMSE |

## 消融实验（Ablation）细节总结

本次批量消融实验（`batch_stage2_h5`）共包含 4 组子实验，各自相对于 Baseline 的设计如下：

1. **`exp_h2_baseline` (基线模型)**:
   - **设计**: 采用最初的原始网络架构（原版 DSTFormer + 原版 DCSA 融合模块），不包含任何 H5 提出的结构改动。
   - **作用**: 作为严格的控制变量基线，反映模型在缺乏时空归纳偏置时的 `no_cond` 性能下限（10.96）。

2. **`exp_h5_no_ltc` (无局部时序卷积)**:
   - **设计**: 在基线上增加了**离散 Token 时空位置编码**、**旋转位置编码 (RoPE)** 以及 **SwiGLU 门控机制**，但**特意去除了**时序前馈网络中的局部时序卷积 (Local Temporal Convolution, LTC)。
   - **作用**: 验证 LTC 在强制模型学习 EMG 连续信号局部平滑性方面的不可替代性。结果表明，即使没有 LTC，仅靠 RoPE 和离散时空编码也能将误差大幅降至 10.50。

3. **`exp_h5_no_rope` (无旋转位置编码)**:
   - **设计**: 在基线上增加了**离散 Token 时空位置编码**、**LTC** 以及 **SwiGLU 门控机制**，但**特意去除了**自注意力机制中的旋转位置编码 (RoPE)。
   - **作用**: 验证 RoPE 在长序列周期性相对运动（如深蹲、跑步）建模上的价值。结果同样降至 10.50，说明 LTC 和离散编码也贡献了巨大的泛化增益。

4. **`exp_h5_full` (H5 完全体)**:
   - **设计**: 集成了所有 H5 改进方案（离散 Token 时空 PE + RoPE + LTC + SwiGLU）。
   - **作用**: 证明各项底层时空归纳偏置相辅相成，组合后能最大化提升模型在 `no_cond` 场景下的空间对齐与时序泛化能力，达到最优性能（10.49）。
